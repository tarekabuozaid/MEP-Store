### تفاصيل الحقول

| الحقل | النوع | مطلوب | سلوك |
|-------|-------|--------|------|
| نوع المعاملة | Select | ✅ | يغير الحقول الظاهرة |
| التاريخ | Date picker | ✅ | Default = اليوم؛ تحذير لو > 30 يوم |
| موقع المخزن | Select | ✅ | مقفل للأمين على مخزنه، منسدل للأدمن |
| الوجهة | Select | ✅ فقط للتحويل | يُخفى لغير التحويل |
| كود الصنف | Text + Autocomplete | ✅ (على الأقل سطر) | اقتراحات تظهر أثناء الكتابة من Master_Items |
| الكمية | Number | ✅ | للصرف: يظهر الرصيد المتاح بجانبه |

### Autocomplete الأصناف

```
المستخدم يكتب: "خرط"
                ↓
[خرطوشة طابعة HP   | ABC-001 | قطعة]
[خرطوشة طابعة Canon| ABC-002 | قطعة]
```

البحث يعمل على: الكود + الاسم معاً

## 4. شاشة مخزوني (Stock View) — الأمين

```
┌─────────────────────────────────────────────────────────────┐
│  مخزون: مدينة زايد                                         │
├─────────────────────────────────────────────────────────────┤
│  🔍 [ ابحث بالاسم أو الكود... ]   [ ⚠️ تحت الحد فقط ]    │
├─────────────────────────────────────────────────────────────┤
│  الكود      │ الاسم              │ الرصيد │ الحد │ الحالة  │
│  ──────────┼───────────────────┼────────┼──────┼─────────│
│  ABC-001   │ خرطوشة HP         │  47    │  10  │  ✅ OK  │
│  ABC-002   │ خرطوشة Canon      │   3    │   5  │  ⚠️ LOW │
│  PPR-A4    │ ورق A4            │   0    │  10  │  🔴 ZERO│
│  ...       │ ...               │  ...   │ ...  │ ...     │
├─────────────────────────────────────────────────────────────┤
│  إجمالي الأصناف: 45   │ تحت الحد: 8   │ نفاد: 2         │
└─────────────────────────────────────────────────────────────┘
```

**الألوان:**
- ✅ رصيد ≥ الحد الأدنى → أخضر
- ⚠️ رصيد < الحد الأدنى وأكبر من 0 → برتقالي
- 🔴 رصيد = 0 → أحمر

## 10. مبادئ التصميم

### الألوان الأساسية

| الاستخدام | اللون |
|-----------|-------|
| الهيدر | `#1a3c5e` (أزرق داكن) |
| الأزرار الرئيسية | `#2563eb` (أزرق) |
| النجاح / OK | `#16a34a` (أخضر) |
| التحذير / LOW | `#d97706` (برتقالي) |
| الخطر / ZERO | `#dc2626` (أحمر) |
| الخلفية | `#f8fafc` |

### متطلبات الاستجابة (Responsive)

| حجم الشاشة | التعديل |
|-----------|---------|
| ≥ 1024px | جدول كامل، كل الأعمدة |
| 768px – 1023px | بعض الأعمدة تُخفى، التنقل أفقي |
| < 768px | الجدول يتحول لـ card layout، نموذج عمودي |

### قواعد UX

1. **لا توجد صفحات منفصلة** — كل شيء في SPA واحدة (Single Page App)
2. **التحميل المرئي** — spinner يظهر أثناء كل اتصال بالـ Apps Script
3. **رسائل النجاح** — Toast notification لمدة 3 ثوانٍ
4. **الـ Autocomplete** يبدأ بعد حرفين
5. **الحقول المطلوبة** علامة `*` حمراء
6. **لا يمكن إرسال النموذج** إلا بعد إصلاح كل الأخطاء المانعة
7. **زر الحفظ** يُعطَّل أثناء الإرسال لمنع النقر المزدوج
