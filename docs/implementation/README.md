## كيفية استخدام هذا التوثيق

```
مطور جديد يبدأ؟
  → 00 ثم 01 ثم 02 ثم 03 ثم 04 ثم 05

تريد بناء شاشة معينة؟
  → 06 للتصميم + 04 للمنطق + 05 للدوال

تريد نشر النظام؟
  → 09 (يحيلك لـ 03 عند إنشاء الجداول)

تريد نقل بيانات Excel؟
  → 10

تريد تنفيذ المشروع على أرض الواقع؟
  → اتبع ترتيب [11-IMPLEMENTATION-PHASES.md](11-IMPLEMENTATION-PHASES.md)
```
